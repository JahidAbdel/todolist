# todolist
TO-DO list faite avec JS et HTML

Le javascript à servi à coder la sauvegarde automatique des donées même en cas
de fermeture de la page web, il a ensuite servi à ajouter les tâches dans la liste
des tâches, puis aussi à modifier l'apparence de la tâche lorsqu'on click dessus 
pour la checker et enfin à créér la croix qui supprime toute une tâche.
